# CQWebSocket SDK 說明文件

## 使用方式

如果你使用 NodeJS，可以通過 NPM 安裝。

```bash
npm install @tsuk1ko/cq-websocket
```

將 SDK 導入代碼

```js
const { CQWebSocket } = require('@tsuk1ko/cq-websocket')
```

或是使用 ES6 import

```js
import { CQWebSocket } from '@tsuk1ko/cq-websocket'
```
